# 学校図書 算数 単元カレンダー案（1・3・6年）（2026-07-16 データ起こし）

学校図書（学図）公式の令和6年度版「みんなと学ぶ 小学校 算数」年間指導計画作成資料（簡易版・Excel）から起こした単元カレンダー。
東書版（spec/東書-算数-単元カレンダー案.md）と同じ様式。既存gen（啓林館ベース）を流用し、時間割だけ差し替える方針。

## 確度メモ

- 単元名・順番・実施月・月別時数: 公式Excel（簡易版）の記載どおり（確度高）
- s の「日」: 月内の累積時数からの比例配分による概算（確度低・目安）。
  月をまたぐ単元は開始月で記載
- 2年・4年・5年は今回の対象外（同じ URL 規則 R6_sansu_nenkan_{学年}_simplified.xlsx で取得可能）
