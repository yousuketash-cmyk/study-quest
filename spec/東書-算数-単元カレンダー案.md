# 東京書籍 算数 単元カレンダー案（1・2・4・5・6年）（2026-07-16 起こし・2026-07-17 に2・4・5年を追記。3年は spec/川口-東書3年-単元カレンダー案.md）

東京書籍公式の令和6年度用「新編 新しい算数」年間指導計画作成資料（略案・令和5年5月31日版）から起こした単元カレンダー。
3年生版（spec/川口-東書3年-単元カレンダー案.md）と同じ様式。既存gen（啓林館ベース）を流用し、時間割だけ差し替える方針。

## 出典（東京書籍公式・原本確認済みのURL）

- 算数：新編 新しい算数 年間指導計画 略案【1年】
  https://ten.tokyo-shoseki.co.jp/text/shou/sansu/data/sansu_keikaku_ryakuan_1.pdf
- 算数：新編 新しい算数 年間指導計画 略案【6年】
  https://ten.tokyo-shoseki.co.jp/text/shou/sansu/data/sansu_keikaku_ryakuan_6.pdf
- どちらも月の欄はなく学期区分＋単元別時数のみ。s は学期内の累積時数からの比例配分（全行）
- 時数の裏取り：1年=3学期制で 1学期41／2学期48／3学期31（総120＋予備16）、
  6年=3学期制で 1学期54／2学期62／3学期35（総151＋予備24）。単元別時数の合計と各学期計が一致することを確認済み
