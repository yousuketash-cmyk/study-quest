# 川口市（東京書籍）3年生 単元カレンダー案（2026-07-16 データ起こし完了・未実装）

東京書籍公式の令和6年度用 年間指導計画作成資料から起こした、川口セット用の単元カレンダー。
着工時はこのファイルの表を index.html の *_KAWA（仮）データに転記する。

## 実装時のメモ

- 器：GRADE_PACKS に「教科書セット」軸を足す（例 D.book="hadano"|"kawaguchi"、既存プレイヤーは hadano 扱い）
- 単元 id は既存と衝突しない接頭辞にする（例 kwu1…, kwc3s1…）。ただし gen/qs は既存を参照共有
- 理社の qs 再割当ては「単元名とsだけ川口版・qs配列は既存参照」で重複データを作らないこと
- 精度に不安が残る箇所：算数の s は比例配分の目安（学校差はペース補正＋手なおしで吸収）
