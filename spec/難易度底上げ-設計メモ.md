# 難易度底上げ（教材の質向上）設計メモ（2026-07-16 合意）

「全体的に問題がやさしい」への対策。実装前の設計図として使う。
着工はこのメモを単一の真実とする。実装は別セッション（並行作業の完了後）。

━━━━━━━━━━━━━━━━━━━━━━
## 0. 基本方針：問題の「2階建て化」

- 問題データに「応用（2階）」の印を付けられる器を作る（全教科で使い回す・1回だけの小工事）
- 既存の教科ごと「ちょうし」自動調整（subjectForm：直近20問の正解率で 🔥/ふつう/🌱）と接続し、
  **🔥好調の子には応用問題を優先して出す**。🌱じっくりの子は基本のまま（やさしさは壊さない）
- 対象学年は 1・3・6 年すべて（学年相応の難しさで）。りか・しゃかいは 3・6 年のみ（1年は教科なし）
- えいごは今回の対象外（据え置き）

## 1. さんすう：応用の「型」を追加

生成式なので型を1つ作れば無限に出題される（費用対効果が最も高い）。

- 2段階で考える文章問題（例：買い物→おつり→分ける）
- 逆算（□を求める。「□×6=42 の □は？」）
- 単位のひっかけ（cm と m、L と dL の混在など）
- 既存の bunRate（🔥=50%）の仕組みに応用型を合流させる

## 2. りか・しゃかい：在庫5倍＋誤答強化

- **在庫5倍**：1単元あたり 5〜7問 → 25〜35問 に増やす
  - 目的は「答えの暗記」による難易度崩壊の防止
  - 規模の目安：6年社会 113問→約560問。3年理科・3年社会・6年理科も同様に×5
- **誤答の紛らわしさ強化**：明らかに変な外れ選択肢を「隣の概念」に差し替える
  （例：正解「酸素」の誤答に「二酸化炭素」「ちっ素」を置く）。既存問題にも適用
- 既存方針の踏襲：教科書丸写し禁止・単元順準拠OK・問題文はオリジナル

## 3. こくご：短文読解の文章問題を新設

技術は既存の4択の仕組みを流用（ほぼゼロ工事）。勝負は良問づくり。

- **問いの型を絞って正解を一意にする**（雑な「なぜ？」は作らない）：
  - 「だれが〜した？」（主語）
  - 「じゅんばんは？」（出来事の順序）
  - 「『それ』は何をさす？」（こそあど言葉）
  - 「気持ちが分かる一文はどれ？」（心情の根拠）
- **文章はすべてオリジナル書き下ろし**（3〜5文。動物・行事など身近な題材。著作権対策）
- **長さは学年別**：1年=2〜3文＋問い1つ／3年=3〜5文＋問い1〜2つ／6年=5〜8文＋問い2つ
- kanjify（習った漢字だけ漢字表示）が全学年対応済みなので、同じ文章が学年ごとに
  適切な表記で表示される。1年生は読み上げボタン併用

## 4. 進め方（2026-07-16 決定：選択肢3の中間方式）

1. 器の小工事（応用の印＋🔥時の優先出題）
2. 誤答の差し替え（りか・しゃかいの既存問題。新規より安い）
3. 中身の一括作成（教科×学年ぶんをまとめて作る）
4. **公開前に「誤り探しの検品」を別セッションで実施**（作った本人以外の目で事実確認・
   正解の一意性チェック・学年相応チェック）→ 合格してから公開
- 分納（単元ごと目視確認）ではなく一括＋検品方式を採用
